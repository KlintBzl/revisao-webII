# ⚜️ Movimento Escoteiro

Landing page desenvolvida para apresentar o Movimento Escoteiro, seus valores e benefícios, além de facilitar o contato de interessados em participar.

O projeto foi criado com foco em **design moderno, responsividade e experiência do usuário**, funcionando em computadores, tablets e celulares.

---

## 🌿 Sobre o projeto

O site tem como objetivo divulgar o escotismo e incentivar a participação de jovens e responsáveis, apresentando informações de forma clara e visual.

A página inclui:

- Seção explicativa sobre o movimento
- Lista de benefícios
- Carrossel de imagens
- Botão de chamada para ação
- Formulário de interesse em modal
- Contato direto via WhatsApp
- Botão de voltar ao topo
- Scroll suave
- Layout totalmente responsivo

---

## 🚀 Tecnologias utilizadas

- HTML5  
- CSS3 (Flexbox + Media Queries)  
- JavaScript (interações e modal)  

---

## 📱 Responsividade

O site foi projetado para funcionar corretamente em:

- 📱 Celulares (até 600px)  
- 📲 Tablets (até 900px)  
- 💻 Desktops  

Inclui reorganização de layout, ajuste de menu e elementos em coluna no mobile.

---

## 🎯 Funcionalidades

✔ Navegação com scroll suave  
✔ Carrossel automático e manual  
✔ Modal com formulário  
✔ Link direto para WhatsApp com mensagem automática  
✔ Botão “voltar ao topo”  
✔ Layout adaptável a diferentes telas
